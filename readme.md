# SW동아리 결과보고서 (Lockey)

**학생들이 6개월간 학습한 내용을 실전적으로 복습하고 응용할 수 있도록, 취약한 사이트를 직접 제작하여 학습한 내용을 바탕으로 정답을 찾게 하였으며, 이를 위해 Docker로 각 문제 환경을 안정적으로 컨테이너화하고 오픈소스 플랫폼인 CTFd를 활용해 직접 CTF 대회를 기획·운영하였습니다.**
